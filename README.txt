
HLL Map-Vote Bot (JSON + RCONv2 + APScheduler)
---------------------------------------------
This zip mirrors the complete scaffold described in the ChatGPT canvas document.

It includes:
- docker-compose.yml
- requirements.txt
- bot/ (Python source)
- data/ (JSON configuration)
- README.md

Unzip into your working directory and run:
    docker compose up -d
or:
    python bot/main.py
